## 🧠 Example Format

### Problem: Two Sum
**Difficulty:** Easy  
**Approach:** Hash Map  
**Time Complexity:** O(n)

```python
def twoSum(nums, target):
    seen = {}
    for i, num in enumerate(nums):
        diff = target - num
        if diff in seen:
            return [seen[diff], i]
        seen[num] = i
